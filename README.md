## Buck-Boost-L-C-Cali
+ 电感计算器
+ 用来计算Buck和Boost的sw端电感值
+ [CSDN](https://blog.csdn.net/m0_49319736/article/details/132533596?spm=1001.2014.3001.5501)
+ 输入相应电压电流参数点击`Cali`即可。
+ 输出电感值`max`和`min`范围内电感均可满足，注意电感的额定电流大小，选取合适电感值。
